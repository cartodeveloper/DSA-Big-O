# BIG O Notation

## Key Concepts

---

- Time Complexity.
- Space Complexity.
- Big O Notation.
- Linear Relationship.
- Constant Time -> O(1)
- Logarithmic Time -> O(log(n))
- Linear Time -> O(n)
- Polynomial Time -> O(n^k)
- Exponential Time -> O(2^n)
